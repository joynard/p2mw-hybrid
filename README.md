# JASTRIP - Personal Shopper & Local Guide Service

An innovative digital platform developed as part of the **Program Pembinaan Mahasiswa Wirausaha (P2MW)** (Student Entrepreneurship Development Program). JASTRIP is designed to help university students find affordable and trustworthy personal shopping (jastip) services and local guides within the campus community.

## Project Overview

JASTRIP is envisioned as a creative economy solution for students, combining the convenience of personal shopping services with student financial independence. The application allows users to request items from specific locations or hire local guides (fellow students) to explore new cities or campuses.

### Key Features:
- **Jastip (Personal Shopper):** Request your favorite items, regional culinary delights, or campus needs from students who are traveling.
- **Local Guide:** Local guiding services by students to help explore cities, campuses, or hidden culinary gems.
- **Booking System:** Easy management of orders for both shopping and guiding services.
- **Profile & Portfolio:** Showcases the reputation of service providers to minimize fraud risks.

## Tech Stack

This application is built using a modern architecture to ensure fast performance and a seamless user experience:

- **Framework:** [Ionic Angular](https://ionicframework.com/) (Standalone Components)
- **Styling:** Vanilla CSS & Ionic Components
- **Icons:** [Ionicons](https://ionicons.com/)
- **Build Tool:** Angular CLI / Webpack

## Getting Started

Ensure you have [Node.js](https://nodejs.org/) and [Ionic CLI](https://ionicframework.com/docs/intro/cli) installed on your system.

1. **Clone this repository:**
   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the project directory:**
   ```bash
   cd P2MW
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the application locally:**
   ```bash
   npm start
   ```
   Or using Ionic CLI:
   ```bash
   ionic serve
   ```
